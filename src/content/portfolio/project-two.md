# Интернет-магазин для локального бизнеса

## О проекте

Разработал интернет-магазин для небольшой компании по продаже handmade товаров в Сен-Назере.

## Функционал

- **Каталог товаров** с фильтрами и поиском
- **Корзина покупок** с сохранением состояния
- **Интеграция платежей** через Stripe
- **Админ-панель** для управления товарами
- **Система заказов** с уведомлениями

## Технологии

- React для фронтенда
- Node.js и Express для бэкенда
- MongoDB для базы данных
- Stripe для обработки платежей

## Результат

Магазин позволил компании расширить географию продаж и увеличить доход на 40% в первые три месяца.
