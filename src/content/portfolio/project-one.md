# Сайт для местного ресторана

## О проекте

Создал полноценный сайт для популярного ресторана в Сен-Назере. Сайт включает в себя:

- **Онлайн-меню** с фотографиями блюд
- **Система бронирования столов**
- **Отзывы клиентов**
- **Интеграция с Instagram и Facebook**
- **Адаптивный дизайн** для всех устройств

## Технологии

- Astro для генерации статического сайта
- Tailwind CSS для стилизации
- JavaScript для интерактивности
- Vercel для хостинга

## Результат

Сайт значительно увеличил количество онлайн-бронирований и помог привлечь новых клиентов из соседних городов.
